== Description ==

This plugin allows user to see how many times a  given page is viewed on each page load. Additional it provides  other features like logged in user view count, reports for admin to do detail analysis

On front end it shows overall view count and view count for logged in user. In Admin it shows all the users with all page and post view count. On second level, it shows all the visit details group by post/page. 

== Installation ==

1. Unzip 'view-count-by-webline.zip' to the '/wp-content/plugins/' directory or add it by 'Add new' in the Plugins menu
2. Activate the plugin through the 'Plugins' menu
3. Go to 'Settings->Page Views' in the admin menu
